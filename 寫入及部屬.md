## 二、部屬步驟

### 1. 推送到 GitHub（若要遠端版控或 Railway）

```powershell
cd c:\Users\user\Desktop\zhe-wei-tech
git remote add origin https://github.com/<你的帳號>/<repo>.git
git branch -M main
git push -u origin main
```

（若已有 remote，改為 `git push origin main`。）

### 2. D 槽部屬（本機熱資料落 D）

```powershell
cd c:\Users\user\Desktop\zhe-wei-tech
powershell -ExecutionPolicy Bypass -File scripts\deploy_to_d_drive.ps1
```

或雙擊 **一鍵部屬到D槽.bat**。

### 3. Railway 部屬（雲端服務）

1. 到 [railway.app](https://railway.app) 新建專案，選 **Deploy from GitHub repo**，連到上述 repo。
2. 在專案 **Variables** 設定：`GEMINI_API_KEY`、`ANTHROPIC_API_KEY` 或 `CLAUDE_API_KEY`（勿提交 .env）。
3. 部署完成後，管理介面：`https://<服務>.up.railway.app/static/index.html` 或 `/admin`；健康檢查：`/health`。

### 4. 本地驗證

```powershell
cd c:\Users\user\Desktop\zhe-wei-tech
python brain_server.py
```

另開終端：`python scripts/test_brain_server.py`。管理介面：http://localhost:8000/admin 。
