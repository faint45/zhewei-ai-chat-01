# 築未科技 — 入口總覽

## 主入口（日常使用）

| 入口 | 用途 |
|------|------|
| **啟動BOT.bat** | 賈維斯 Discord Bot（日常對話）|
| **啟動_築未科技大腦_完整版.bat** | 主腦（8 階段 Agent、工具執行）|
| **啟動_PM2.bat** / **停止_PM2.bat** | PM2 常駐模式 |
| **一鍵同步至雲端.bat** | 備份至雲端 |
| **開啟賈維斯控制台.bat** | 開啟主腦網頁介面 |

## 維護

| 入口 | 用途 |
|------|------|
| **合併到D槽.bat** | 合併桌面版到 D 槽 |
| **清理無用資料.bat** | 清理快取、tmp |
| **歸檔非主入口.bat** | 將非主入口移至 archive |
