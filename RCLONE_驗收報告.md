## 六、使用性驗收

### 6.1 手動掛載流程

```
1. 右鍵「挂载 Google Drive 为 Z 槽.bat」
2. 選擇「以系統管理員身分執行」
3. 保持視窗開啟
4. Z 槽出現並可使用
```

### 6.2 卸載流程

```
1. 執行「卸载 Google Drive.bat」
   或
2. 關閉掛載視窗
3. Z 槽自動卸載
```

### 6.3 測試流程

```

→ 測試 rclone 版本
```
