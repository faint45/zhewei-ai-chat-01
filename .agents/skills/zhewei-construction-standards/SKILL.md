# 築未科技 — 營造管理規範技能

## 適用情境

- 產出或檢查與**公共工程品質管理**相關的文案、查核表、表單。
- 引用或對齊**交通部標線規範**、施工標準、查驗要項。
- 撰寫工地管理、品管流程、文件歸檔邏輯時需符合**國內法規與實務**。

## 規範來源（可擴充）

- 台灣公共工程品質管理手冊
- 交通部標線規範與相關施工綱要
- 營造業法及施行細則、品管相關子法
- 專案內既有規定：見 `.cursor/rules/架構守則.mdc`、`全系統架構鎖定規則.md`

## 使用原則

1. **先確認任務是否落在營造/品質/標線/法規**：是則套用本技能，否則不載入細節以節省 Token。
2. **產出須標註依據**：引用規範時註明出處（如「依公共工程品質管理手冊第 x 章」）。
3. **與架構守則並存**：資料分流（D/Z）、報表路徑、LaTeX 進度格式等仍依架構守則，本技能不覆寫技術架構。

## 進階內容（Level 2/3）

團隊可將具體條文、查核表範本、標線圖說等放入本技能之 `references/`，並在 SKILL.md 內以簡短條列指向，供需要時才載入。
