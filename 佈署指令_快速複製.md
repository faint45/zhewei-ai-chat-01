# 築未大腦 — 佈署指令（快速複製）

## 方式一：本機僅大腦（診斷後啟動）

```powershell
cd c:\Users\user\Desktop\zhe-wei-tech
python startup_diagnostics.py
python brain_server.py
```

---

## 方式二：Docker Compose（大腦 + Cloudflare Tunnel）

先確認 `.env` 有 `CLOUDFLARE_TOKEN`，且 `D:/brain_workspace`、`Z:/Zhewei_Brain` 可存取。

```powershell
cd c:\Users\user\Desktop\zhe-wei-tech
docker compose up -d
```

---

## 方式三：Windows 服務（開機自啟）

**請以系統管理員身分執行**：

```batch
安装築未大腦服務.bat
```

（在檔案總管對 `安装築未大腦服務.bat` 右鍵 → 以系統管理員身分執行，或於管理員 CMD 中執行上述指令。）
