## 鋼筋施工規範摘要

### 搭接長度（CNS 560 / ACI 318）

- **一般搭接**: ≥ 40d（d = 鋼筋直徑）
- **受拉區 A 級搭接**: 1.0 × Ld
- **受拉區 B 級搭接**: 1.3 × Ld
- **受壓搭接**: ≥ 30d 且 ≥ 30cm

### 保護層厚度

- **基腳、地梁**（直接接觸土壤）: ≥ 7.5cm
- **牆、梁、柱**（室外暴露）: ≥ 4cm
- **樓板、牆**（室內）: ≥ 2cm
- **預力構件**: ≥ 4cm

### 間距要求

- 最小淨距 ≥ 鋼筋直徑 或 2.5cm（取大值）
- 粗骨材最大粒徑的 4/3 倍
- 同層鋼筋淨距 ≥ d 或 2.5cm

### 彎鉤規定

- **標準彎鉤**: 180° + 4d 延伸 或 90° + 12d 延伸
- **箍筋彎鉤**: 135° + 6d 延伸（耐震設計）
- **繫筋彎鉤**: 90° + 6d 延伸

### 常見缺失

1. 搭接位置集中在同一斷面（應錯開 ≥ 25d 或 ≥ 40cm）
2. 保護層不足（墊塊缺漏或位移）
3. 箍筋間距過大（柱端加密區應 ≤ 10cm）
4. 搭接長度不足（特別是 D25 以上大號鋼筋）
