### 路由器DDNS配置步驟

#### 1. 登錄路由器管理界面
- 地址：`http://192.168.1.1`
- 用戶名/密碼：路由器背面標籤

#### 2. 找到DDNS設置
不同品牌位置：
- **TP-Link**: 網絡參數 → DDNS
- **ASUS**: 高級設置 → 外部網絡 → DDNS
- **Netgear**: 高級 → 高級設置 → 動態DNS
- **D-Link**: 工具 → 動態DNS

#### 3. 配置DDNS參數
以 **DuckDNS** 為例：

```
服務提供商: DuckDNS (或選擇自定義)
主機名稱: zhuwei-ai.duckdns.org
用戶名/密碼: 留空
Token/Key: 您的DuckDNS Token
更新間隔: 10分鐘
```

#### 4. 啟用並保存

### 路由器DDNS vs 本地腳本對比

| 特性 | 路由器DDNS | 本地腳本 |
|------|-----------|----------|
| **穩定性** | 更高（路由器重啟自動恢復） | 依賴電腦開機 |
| **更新速度** | 更快（路由器直接處理） | 稍慢（需要電腦運行） |
| **配置複雜度** | 中等（需要路由器支持） | 簡單（只需運行腳本） |
| **兼容性** | 依賴路由器型號 | 所有Windows電腦 |

## 🔧 故障排除

### 常見問題

**Q: DDNS更新失敗？**
A: 檢查：
1. Token是否正確
2. 網絡連接是否正常
3. 域名格式是否正確

**Q: 無法通過域名訪問？**
A: 檢查：
1. 端口轉發是否配置正確
2. 防火牆是否允許端口8000
3. DDNS域名解析是否正確

**Q: IP更新不及時？**
A: 縮短更新間隔到5-10分鐘
