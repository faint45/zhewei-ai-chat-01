## 🚀 快速部署

### 方法一：Netlify（推薦）
1. 訪問 [netlify.com](https://netlify.com)
2. 點擊 "GitHub" 登入
3. 選擇 "New site from Git"
4. 選擇此倉庫
5. 自動部署完成

### 方法二：Vercel
1. 訪問 [vercel.com](https://vercel.com)
2. 點擊 "Continue with GitHub"
3. 導入此倉庫
4. 自動部署

### 方法三：Cloudflare Pages
1. 訪問 [pages.cloudflare.com](https://pages.cloudflare.com)
2. 連接 GitHub 帳號
3. 選擇此倉庫部署
