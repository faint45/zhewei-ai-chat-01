| 路徑 | 說明 |

**建議**：移入 `archive/agents_legacy/` 保留備查。

### 4. root 層 discord_bot.py（遠程控制）

| 路徑 | 說明 |
|------|------|
| `discord_bot.py` | ZhuWeiTechDiscordBot，遠程控制用 |

**說明**：目前一鍵啟動使用 `Jarvis_Training/jarvis_discord_bot.py`。`discord_bot.py` 僅在 `ops/legacy_launchers` 與舊文件中被引用。  
**建議**：移入 `archive/legacy_discord_bot.py`，若需遠程控制可再還原。

### 5. MetaGPT 環境缺失

| 路徑 | 說明 |
|------|------|
| `metagpt/` | 工具 schemas，仍被使用 |
| `MetaGPT_env/` | **不存在** |

**說明**：`jarvis_discord_bot.py` 可選擇 `metagpt` executor，但 `MetaGPT_env/.venv/Scripts/metagpt.exe` 不存在會導致失敗。  
**建議**：保留 `metagpt/`，在文件中註明需另行安裝 MetaGPT；或移除 metagpt 相關選項。
