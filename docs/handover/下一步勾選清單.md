# 下一步勾選清單

做完 PR 後，剩下兩項可依序勾選完成。

---

## 一、完整跑八階段

- [ ] 在 `.env` 填 `GEMINI_API_KEY=`（從 https://aistudio.google.com/app/apikey 取得）
- [ ] 本機啟動 Ollama（開始選單或 `ollama serve`）
- [ ] 雙擊 **啟動完整跑.bat** 或執行 `python scripts\run_full_system.py`
- [ ] 開 http://localhost:8000/admin 或 /chat 測試八階段

---

## 二、外網 brain.zhe-wei.net

- [ ] 到 Cloudflare Zero Trust → Tunnels → 複製「完整 Token」（eyJ... 開頭，非 UUID）
- [ ] 在 `.env` 把 `CLOUDFLARE_TOKEN=` 改為該完整 Token
- [ ] 執行：`docker compose down` 再 `docker compose up -d`
- [ ] 確認 `docker compose ps` 中 tunnel 為 **Up**，再開 https://brain.zhe-wei.net/health 測試
