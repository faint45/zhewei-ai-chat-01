## 一、專案總覽

### 1.1 專案定位

築未科技大腦是一個**個人 AI 代理人系統**，主要功能：

- ✅ **遠端對話**：從任何設備（PC、iOS 手機）與大腦對話
- ✅ **本地模型**：使用 Ollama 本地模型進行編碼、創作
- ✅ **任務分配**：智能分配任務給不同的 AI 模型
- ✅ **遠程控制**：通過 SSH 和 Web 介面遠程管理主機
- ✅ **監控面板**：實時監控 API 調用和系統狀態

**核心價值：**
- 🤖 私人 AI 助手，數據不離開本地
- 🌐 隨時隨地訪問，通過 Tailscale 免費內網穿透
- 🔒 端到端加密，安全可靠
- ⚡ 快速響應，本地模型延遲 < 2 秒
