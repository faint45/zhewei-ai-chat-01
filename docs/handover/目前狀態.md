## 二、你要做的（對照）

| 項目 | 你要做的 |
|------|----------|
| **git push** | 執行：`scripts\git_push.bat https://github.com/你的帳號/你的repo.git`（把 URL 換成你的）。 |
| **完整跑八階段** | 在 .env 填入 `GEMINI_API_KEY=你的key`；本機啟動 Ollama；之後用 **啟動完整跑.bat** 或 `python scripts/run_full_system.py`。 |
| **外網 brain.zhe-wei.net** | 在 .env 將 `CLOUDFLARE_TOKEN` 改為**完整 Token**（Zero Trust → Tunnels → 你的隧道 → Install connector 或 Reinstall → 複製整段），儲存後執行 `docker compose up -d`。 |
