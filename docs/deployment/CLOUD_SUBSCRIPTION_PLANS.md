## 🔵 阿里雲訂閱方案

### 方案 A：輕量應用伺服器（推薦新手）

**適合：個人開發者、小型專案、測試環境**

#### 基礎配置
```
產品名稱：輕量應用伺服器
規格：2核4GB
儲存：40GB SSD
帶寬：3 Mbps 固定帶寬
流量：1 TB/月
作業系統：Ubuntu 22.04 LTS
```

#### 價格方案
| 付費方式 | 價格 | 優惠 | 總成本 |
|---------|------|------|--------|
| **月付** | ¥108/月 | - | ¥1,296/年 |
| **年付** | ¥90/月 | 節省 17% | ¥1,080/年 |
| **學生價** | ¥9.5/月 | 節省 91% | ¥114/年 |

#### 購買步驟
1. **登入阿里雲控制台**
   - 網址：https://www.aliyun.com
   - 註冊/登入帳號

2. **選擇產品**
   - 產品 → 計算 → 輕量應用伺服器
   - 或直接搜尋「輕量應用伺服器」

3. **配置選擇**
   ```
   地域：華東1（杭州）或 華北2（北京）
   鏡像：系統鏡像 → Ubuntu 22.04
   套餐：2核4GB 3M
   購買時長：12個月（推薦）
   數量：1台
   ```

4. **網路配置**
   - 帶寬：3 Mbps（固定）
   - 流量包：1 TB/月
   - 公網 IP：自動分配

5. **安全組設定**
   ```
   開放端口：
   - 22 (SSH)
   - 80 (HTTP)
   - 443 (HTTPS)
   - 8000-9000 (應用端口)
   ```

6. **確認訂單**
   - 檢查配置
   - 選擇付費方式（年付最划算）
   - 使用代金券（如有）
   - 完成支付

#### 學生認證流程
1. **申請學生認證**
   - 進入「學生專區」
   - 上傳學生證或在學證明
   - 等待審核（通常 1-2 工作天）

2. **購買學生機**
   - 認證通過後
   - 選擇「學生輕量應用伺服器」
   - ¥9.5/月（首年）
   - 限購 1 台

### 方案 B：ECS 雲伺服器（推薦企業）

**適合：企業應用、生產環境、需要更多彈性**

#### 基礎配置
```
產品名稱：雲伺服器 ECS
實例規格：ecs.t6-c1m2.large
規格：2核4GB
儲存：40GB ESSD 雲盤
帶寬：按使用流量計費（建議設定 5 Mbps 峰值）
作業系統：Ubuntu 22.04 LTS
```

#### 價格方案
| 付費方式 | 價格 | 流量費 | 總成本 |
|---------|------|--------|--------|
| **按量付費** | ¥0.32/小時 | ¥0.8/GB | 約 ¥230/月 |
| **月付** | ¥138/月 | ¥0.8/GB | 約 ¥180/月 |
| **年付** | ¥108/月 | ¥0.8/GB | 約 ¥150/月 |

#### 購買步驟
1. **登入 ECS 控制台**
   - 產品 → 計算 → 雲伺服器 ECS
   - 點擊「建立實例」

2. **基礎配置**
   ```
   計費方式：包年包月
   地域：華東1（杭州）
   可用區：隨機分配
   實例規格：
     - 規格族：通用型 t6
     - 實例規格：ecs.t6-c1m2.large（2核4GB）
   ```

3. **鏡像選擇**
   ```
   鏡像類型：公共鏡像
   作業系統：Ubuntu 22.04 64位
   ```

4. **儲存配置**
   ```
   系統盤：
     - 雲盤類型：ESSD 雲盤
     - 容量：40 GB
   資料盤：暫不添加（可後續擴充）
   ```

5. **網路配置**
   ```
   網路類型：專有網路 VPC
   公網 IP：分配公網 IPv4 地址
   帶寬計費方式：按使用流量
   峰值帶寬：5 Mbps
   ```

6. **安全組**
   ```
   選擇安全組：新建安全組
   規則：
     - SSH (22)
     - HTTP (80)
     - HTTPS (443)
     - 自訂 TCP (8000-9000)
   ```

7. **系統配置**
   ```
   登入憑證：自訂密碼
   實例名稱：zhewei-brain-prod
   主機名稱：zhewei-brain
   ```

8. **確認訂單**
   - 購買時長：12個月
   - 自動續費：開啟（可選）
   - 完成支付

### 方案 C：阿里雲 AI 服務整合

**適合：需要使用通義千問等 AI 服務**

#### 基礎服務
```
雲伺服器：2核4GB（¥108/月）
+ 通義千問 API
+ OSS 物件儲存
+ SLS 日誌服務
```

#### AI API 價格（通義千問）
| 模型 | 輸入價格 | 輸出價格 | 適用場景 |
|------|---------|---------|---------|
| **qwen-turbo** | ¥0.008/千tokens | ¥0.008/千tokens | 日常對話 |
| **qwen-plus** | ¥0.02/千tokens | ¥0.02/千tokens | 複雜任務 |
| **qwen-max** | ¥0.12/千tokens | ¥0.12/千tokens | 專業應用 |

#### 月度預估成本
```
雲伺服器：¥108/月
通義千問 API：¥50-200/月（根據使用量）
OSS 儲存：¥1-5/月（10GB 內）
SLS 日誌：¥0-10/月（1GB 內免費）
─────────────────────────
總計：¥159-323/月
```

#### 購買步驟
1. **開通通義千問**
   - 進入「模型服務靈積」
   - 選擇「通義千問」
   - 開通服務（按量付費）
   - 獲取 API Key

2. **開通 OSS**
   - 產品 → 儲存 → 物件儲存 OSS
   - 建立 Bucket
   - 選擇地域（與 ECS 同地域）
   - 儲存類型：標準儲存

3. **配置環境變數**
   ```bash
   # 在 .env 檔案中添加
   DASHSCOPE_API_KEY=sk-xxxxxxxxxxxxx
   ALIYUN_OSS_ENDPOINT=oss-cn-hangzhou.aliyuncs.com
   ALIYUN_OSS_BUCKET=zhewei-brain
   ALIYUN_OSS_ACCESS_KEY_ID=LTAI5xxxxxxxxxxxxx
   ALIYUN_OSS_ACCESS_KEY_SECRET=xxxxxxxxxxxxxx
   ```

## 💡 常見問題

### Q1: 輕量應用伺服器和雲伺服器有什麼區別？

**輕量應用伺服器：**
- ✅ 價格便宜
- ✅ 配置簡單
- ✅ 適合小型應用
- ❌ 彈性較低
- ❌ 功能較少

**雲伺服器（ECS/CVM）：**
- ✅ 功能完整
- ✅ 彈性擴展
- ✅ 適合企業應用
- ❌ 價格較高
- ❌ 配置複雜

**建議：**
- 個人/小型專案 → 輕量應用伺服器
- 企業/生產環境 → 雲伺服器

### Q2: 騰訊雲和阿里雲該選哪個？

**選騰訊雲如果：**
- 預算有限（便宜 ¥18/月）
- 需要整合微信/QQ
- 需要更多流量（2TB vs 1TB）
- 開發小程式或遊戲

**選阿里雲如果：**
- 企業級應用
- 需要整合淘寶/釘釘
- 需要更完善的文件
- 需要更好的技術支援

**建議：**
- 個人開發者 → 騰訊雲（性價比高）
- 企業用戶 → 阿里雲（穩定性好）

### Q3: 學生優惠有什麼限制？

**騰訊雲學生優惠：**
- 需要學生認證（學生證或教育郵箱）
- 限購 1 台
- 首年 ¥10/月，次年恢復原價
- 畢業後失效

**阿里雲學生優惠：**
- 需要學生認證（學生證或在學證明）
- 限購 1 台
- 首年 ¥9.5/月，次年恢復原價
- 畢業後失效

### Q4: 可以隨時升級配置嗎？

**輕量應用伺服器：**
- ✅ 可以升級套餐
- ❌ 不能降級
- ✅ 可以擴充儲存
- ❌ 不能更換地域

**雲伺服器：**
- ✅ 可以升級/降級配置
- ✅ 可以擴充儲存
- ✅ 可以調整帶寬
- ❌ 不能更換地域

### Q5: 流量用完了怎麼辦？

**輕量應用伺服器：**
- 流量用完後會限速（1 Mbps）
- 不會額外收費
- 下個月自動恢復

**雲伺服器（按流量計費）：**
- 按實際使用量計費（¥0.8/GB）
- 可以設定流量告警
- 建議設定帶寬上限
