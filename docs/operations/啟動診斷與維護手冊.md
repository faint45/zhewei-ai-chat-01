# 🧪 築未科技：啟動診斷與維護手冊

## 📌 診斷檢查項目說明

本系統採用「核心必備」與「可選增強」雙軌制檢查。

### 必要項目（必須通過）

- **Z 槽 (Google Drive)**：系統的「海馬迴」，用於存儲日報表與長期記憶。若顯示 ❌，請確認 Rclone 是否已連線。
- **Python 環境**：主系統需運行於 Python 3.10 以上（建議 3.12+，含 3.14）。
- **核心檔案**：包含決策引擎 (`agent_logic.py`)、AI 服務 (`ai_service.py`) 與通訊伺服器 (`brain_server.py`)。
- **依賴**：FastAPI、uvicorn（WebSocket 伺服器必要）。

### 可選項目（不影響核心功能）

- **視覺環境 (venv_vision)**：專為 Python 3.12 建立的 AI 視覺區。若顯示 ⚠️，則無法執行 YOLO 影像辨識。
- **GPU 加速 (CUDA)**：偵測 **RTX 4060 Ti** 是否已載入 PyTorch 加速。
- **目錄完整性**：檢查 `input/`、`processed/`、`models/` 等緩存目錄是否存在（位於 `brain_workspace` 或 `D:\brain_workspace`）。
- **Port 8000**：檢查是否已被佔用；可設定環境變數 `BRAIN_WS_PORT` 改用其他埠。

## 🛠️ 故障排除

1. **Z 槽錯誤**：請手動執行 Rclone 掛載指令或檢查網路連線。可參考專案內 `RCLONE_交接說明.md`。
2. **系統錯誤（iPhone 顯示）**：請回主機查看啟動視窗的 Log 紀錄；日誌路徑可由 `BRAIN_LOG_FILE` 設定（預設 `D:/brain_workspace/brain_system.log`）。
3. **GPU 未偵測**：請確認 `venv_vision` 環境下的 `torch` 版本是否為 `cu121` 結尾。
4. **診斷腳本**：執行 `python startup_diagnostics.py` 或雙擊 `系統環境診斷.bat`；完整啟動請使用 `AI大腦總指揮部.bat`。
