# 築未科技系統 - 快速啟動指南

## 🚀 啟動方式

### 方式1：使用一鍵啟動腳本（推薦）

雙擊運行以下腳本即可啟動所有服務：

```batch
start_all_services.bat
```

此腳本會自動：
- 啟動網站服務器（端口 8000）
- 啟動遠程控制服務器（端口 8005）
- 顯示所有訪問地址

---

### 方式2：手動啟動各個服務

打開多個終端窗口，分別執行以下命令：

**終端窗口1 - 啟動網站服務器：**
```bash
python website_server.py
```

**終端窗口2 - 啟動遠程控制服務器：**
```bash
python remote_control_server.py
```

---

## 📋 訪問地址

啟動完成後，您可以在瀏覽器中訪問以下地址：

| 服務 | 端口 | 訪問地址 | 說明 |
|------|------|---------|------|
| **網站首頁** | 8000 | http://localhost:8000 | 企業官網主頁 |
| **AI對話** | 8000 | http://localhost:8000/chat | AI大腦智能對話系統 |
| **管理後台** | 8000 | http://localhost:8000/admin | 管理員登入和儀表板 |
| **聯繫我們** | 8000 | http://localhost:8000/contact | 在線諮詢表單 |
| **遠程控制** | 8005 | http://localhost:8005 | 遠程控制本地AI模型 |

---

## 🔧 系統測試

運行測試腳本檢查所有組件是否正常：

```bash
python test_all.py
```

或者雙擊運行：

```batch
test_system.bat
```

---

## ⚠️ 注意事項

1. **端口占用**
   - 確保端口 8000 和 8005 未被其他程序占用
   - 如果端口被占用，請修改對應服務器文件的端口配置

2. **依賴套件**
   - 確保已安裝所有依賴套件：
   ```bash
   pip install fastapi uvicorn flask requests
   ```

3. **AI模型**
   - AI功能需要配置 Ollama 或其他 AI 模型服務
   - 詳見 `config_ai.py` 配置文件

4. **資料庫**
   - 資料庫文件 `website.db` 會在首次啟動時自動創建

---

## 🛑 停止服務

在對應的終端窗口中按 `Ctrl + C` 即可停止服務。

---

## 📞 技術支持

如遇到問題，請檢查：
1. 終端窗口中的錯誤日誌
2. 瀏覽器開發者工具（F12）中的控制台錯誤
3. 運行 `python test_all.py` 進行系統診斷
