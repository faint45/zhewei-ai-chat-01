## 故障排查

### 問題 1：無法連接

**症狀：**
```
Safari 顯示"無法打開頁面"
```

**解決方法：**

```
1. 檢查 Tailscale 是否已登入
   - 打開 Tailscale App
   - 檢查狀態是否為"已連接"

2. 檢查 VPN 連接
   - iOS 設置 > VPN 與設備管理
   - 確認 Tailscale VPN 已啟用

3. 檢查網絡
   - 確認 iOS 設備已連接到網絡
   - 嘗試訪問其他網站

4. 重試
   - 關閉 Safari
   - 重新打開
   - 輸入地址：http://100.116.133.23:8000
```

### 問題 2：頁面空白

**症狀：**
```
頁面加載成功但顯示空白
```

**解決方法：**

```
1. 清除 Safari 快取
   - 設置 > Safari > 清除歷史記錄與網站資料
   - 選擇"清除"
   - 選擇"過去 1 小時"

2. 重新加載頁面
   - 下拉刷新

3. 檢查 JavaScript
   - 設置 > Safari > 進階
   - 啟用"JavaScript"
```

### 問題 3：回應慢

**症狀：**
```
發送消息後，回應需要很長時間
```

**可能原因和解決方法：**

```
1. Ollama 模型運行慢
   - 在主機上檢查 Ollama 狀態
   - 考慮換用更小的模型

2. 網絡延遲
   - 檢查 Tailscale 連接質量
   - 切換到更好的網絡

3. 主機負載高
   - 在主機上檢查 CPU/記憶體使用率
   - 關閉其他不必要的程序
```

### 問題 4：界面顯示異常

**症狀：**
```
排版亂、按鈕顯示不正確
```

**解決方法：**

```
1. 切換到桌面網站模式
   - Safari 地址欄 > AA 按鈕
   - 請求桌面網站

2. 調整 Safari 設置
   - 設置 > Safari
   - 網站設置 > 顯示桌面網站：關閉
```
