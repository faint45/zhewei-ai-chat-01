# 築未科技系統 - 使用手冊

## 📦 系統組件

### 核心服務
- **website_server.py** - 網站服務器（端口 8000）
- **remote_control_server.py** - 遠程控制服務器（端口 8005）
- **ai_service.py** - AI 服務模組
- **config_ai.py** - AI 配置模組

### 前端組件
- **templates/chat.html** - AI 對話頁面
- **templates/contact.html** - 聯繫我們頁面
- **templates/remote_control.html** - 遠程控制頁面
- **static/css/admin.css** - 管理後台樣式（已修復滾動條）

---

## 🚀 快速啟動

### 方式1：一鍵啟動（推薦）
雙擊 `start_all_services.bat`

### 方式2：分別啟動
雙擊以下腳本：
- `啟動網站服務器.bat` - 啟動網站服務
- `啟動遠程控制服務器.bat` - 啟動遠程控制

### 方式3：命令行啟動
```bash
# 終端窗口1
python website_server.py

# 終端窗口2
python remote_control_server.py
```

## ⚠️ 常見問題

### Q: 端口被占用怎麼辦？
A: 修改服務器文件中的端口配置，或停止占用端口的程序

### Q: AI 功能無法使用？
A: 檢查 `config_ai.py` 中的 AI 模型配置，確保 Ollama 或其他 AI 服務正常運行

### Q: 無法訪問管理後台？
A: 確保資料庫中有管理員賬號，或檢查登入認證邏輯

### Q: 頁面樣式異常？
A: 清除瀏覽器緩存，或檢查 `static/css/` 目錄下的樣式文件
