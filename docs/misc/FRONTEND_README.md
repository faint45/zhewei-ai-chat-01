# 築未科技 AI 對話系統 - 前端部署

這個目錄包含靜態前端文件，可以部署到 Vercel 或其他靜態網站平台。

## 部署選項

### 1. Vercel 部署（推薦）

```bash
# 安裝 Vercel CLI
npm i -g vercel

# 部署
vercel
```

或直接通過 Vercel Dashboard 連接此 GitHub 仓库並部署。

### 2. Railway 部署

 Railway 已配置為提供靜態文件服務。更新 `index.html` 後，Railway 會自動重新部署。

- 🧠 AI 對話系統
